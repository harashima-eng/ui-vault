06 — Pin + Fade Sequence

01

Define the Trigger

Every ScrollTrigger animation starts with a trigger element. When it enters the viewport boundary you define, the animation begins. Set start and end to control precisely when.

02

Pin the Container

Setting pin: true locks the element in place while the user scrolls. The page adds padding below automatically so content flow isn't broken. Ideal for step-by-step reveals.

03

Scrub the Timeline

With scrub: true, the animation progress maps directly to scroll position. Scroll forward to advance, scroll back to reverse. The animation becomes a scroll-driven video.

04

Ship with Confidence

ScrollTrigger handles resize, refresh, and cleanup automatically. It calculates positions on the fly, works with dynamic content, and respects prefers-reduced-motion.
